# Support

## What This Project Supports

`cobol-intel` is maintained as an actively developed engineering project for
COBOL analysis, documentation, and modernization workflows.

## Where To Ask For Help

- Usage questions: open a GitHub issue with reproduction steps and expected behavior.
- Bug reports: use the bug report issue template.
- Feature requests: use the feature request issue template.
- Security concerns: follow [SECURITY.md](SECURITY.md) instead of filing a public issue.

## What To Include In Support Requests

- operating system and Python version
- installation method
- command used
- sample input or minimal reproduction
- expected result and actual result
- relevant logs, stack traces, or artifact paths

## Support Boundaries

- Best-effort support is provided for the latest released version and `main`.
- Custom enterprise integration work is out of scope for free issue support.
- GPU fine-tuning support is limited to reproducible issues in the published tooling.
